Buttons


Overview & Usage

There will only be three buttons in this design system. A main button, an alternative color depending on background, and a ghost button for secondary CTAs. Buttons should be used to guide the user to a place that they may want to go to or need to go.


Accessibility

The red buttons should be used most often barring significant background changes with which the black button can be used. Avoid using the ghost button on backgrounds that make it difficult to read.
