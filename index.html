Vince Laylo Design System

This is the design system for Vince Laylo's Website (WIP). The content seen in this documentation provides a general styleguide for this website, as well as other Vince Laylo products--it will show how the brand shall be styled. This design system was created using Bootstrap.


Identity: Elegant, Simple, and Clean

Design decisions were made in order to create a brand that is simple, yet elegant. I want my brand to exude a sense of professionalism yet maintain an approachable and friendly identity.
